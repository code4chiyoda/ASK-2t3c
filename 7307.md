# Code for Chiyoda と、その構成員が交わす契約（憲法）

## 要約

以下は、Code for Chiyoda 全体と、その構成員が交わす契約です。この文書では憲法を扱います。

## 憲法

**われわれは、民主主義を根幹として、左右いずれにも偏らず、平和を愛し、自由、真理の探求を是とする。シビックハッカーの自由が侵されるとき、われわれは団結して、あくまで自由を守る。**

## 推奨事項

構成員は憲法を守るためにとれる可能な選択肢をじゅうぶんに検討しなければなりません。

規則やガイドラインが作られた場合、憲法と矛盾することが考えられます。しかし、それは予算や納期の都合で突貫工事をしたことによる不具合です。理想像のために、常にリファクタリングされ続けるべきです。
