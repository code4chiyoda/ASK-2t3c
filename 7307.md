# Code for Chiyoda と、その構成員が交わす契約（憲法）

## 要約

以下は、Code for Chiyoda 全体と、その構成員が交わす契約です。この文書では憲法を扱います。

## 集団の三原則

1. 他人を尊重できない者は、出ていけ。
2. われわれは、民主主義を根幹として、左右いずれにも偏らず、平和を愛し、自由、真理の探求を是とする。
3. シビックハッカーの自由が侵されるとき、われわれは団結して、あくまで自由を守る。

## 検討の三原則

1. 問題が発生した場合には、個人判断によって処理することなく、全構成員に合理的な手段によって告知し、全員で検討する。
2. 構成員が制約された状況の中で判断するのではなく、市民の広範な意見を聞く。
3. とりわけ人権侵害に関わる問題については、偏見と予断にとらわれないよう、問題の当事者に聞く。

## 推奨事項

構成員は憲法を守るためにとれる可能な選択肢をじゅうぶんに検討しなければなりません。

規則やガイドラインが作られた場合、憲法と矛盾することが考えられます。しかし、それは予算や納期の都合で突貫工事をしたことによる不具合です。理想像のために、常にリファクタリングされ続けるべきです。
